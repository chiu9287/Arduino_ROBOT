# Arduino_ROBOT
黃威仁實驗室_田間機器人

此處存放機器人相關零件資料、製作紀錄以及程式碼，供日後學弟妹製作使用。

## 檔案架構

```
Arduino_ROBOT
├─基本功能

│  │  └─main
│  └─零件相關資料

│      │  └─程式
│      ├─HX711(電子秤)
│      │  └─程式
│      ├─L298N(燈條控制器)
│      │  └─程式
│      ├─L298N(馬達控制)
│      │  └─程式
│      ├─laser(雷射)
│      │  └─程式
│      ├─LCD_I2C(LCD顯示器)
│      │  └─程式
│      ├─MG_996R(伺服馬達)
│      │  └─程式
│      ├─MPU_6050(陀螺儀)
│      │  └─程式
│      ├─RGB_LED(LED燈泡)
│      │  └─程式
│      ├─SR04(超音波測距)
│      │  └─程式

```
